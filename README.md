# Anubhavaḥ — अनुभवः Experience Centre

A white-label, identity-masking **Experience Centre Portal** built for **DT Solution**.

## Overview

अनुभवः (Anubhavaḥ) is a private portal that lets clients and agencies access curated digital experiences using unique IDs — while hiding the identity chain between vendor, agency, and end client.

## ID System

| ID | Type | Access |
|---|---|---|
| `CL22MDL` | Client | DT Solution exp centre → Medialoop website |
| `AG23MDL` | Agency | Medialoop workspace with all projects |
| `CL31BBM` | Client | Medialoop exp centre → BBM website |

## Tech Stack

- Pure HTML + CSS + Vanilla JS (zero dependencies)
- Hash-based SPA routing
- Google Fonts: Noto Sans Devanagari + Outfit + Inter

## Features

- 🔒 Identity chain hiding
- 📱 Device preview modes (Desktop, MacBook, iPad, iPhone)
- 🎨 Per-brand theming (DT Solution: black+red, Medialoop: dark+pink)
- 🖥️ Full screen modal preview
- ✅ Delivered vs Active project separation
